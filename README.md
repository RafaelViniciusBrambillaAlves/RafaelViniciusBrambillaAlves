## 🚀 Projetos

Aqui estão alguns dos meus projetos mais recentes!

### Backend
- **[PicPay Simplificado (Desafio Backend)](https://github.com/RafaelViniciusBrambillaAlves/picpay-challenge)**  
  Plataforma de pagamentos com regras de negócio para múltiplos tipos de usuário (comum e lojista), incluindo transferências e controle de saldo.  
  Desenvolvido com foco em **Clean Architecture, testes automatizados e boas práticas de mercado**.  
  Stack: .NET 8, C#, SQL Server, Entity Framework, Docker, JWT, Swagger

- **[Sistema de Passe de Transporte](https://github.com/RafaelViniciusBrambillaAlves/desafio-backend-pref-rio)**  
  API completa para gestão de transporte digital, com foco em **Clean Architecture, SOLID e autenticação OAuth2/JWT**.  
  Stack: FastAPI, MongoDB, MinIO, Docker
  
- **[API Mottu](https://github.com/RafaelViniciusBrambillaAlves/api-mottu)**  
  API robusta para gerenciamento de motos, entregadores e locações, utilizando **mensageria com Kafka e persistência relacional**.  
  Stack: FastAPI, PostgreSQL, Kafka, MinIO, Docker
  
### Engenharia de dados

- **[Data Lakehouse](https://github.com/RafaelViniciusBrambillaAlves/data-lakehouse)**  
  Projeto end-to-end cobrindo todo o ciclo do dado: ingestão, processamento e análise.  
  Foco em **arquitetura moderna de dados e escalabilidade**.  
  Stack: PySpark, Kafka, Airflow, PostgreSQL, MinIO, Docker
  
- **[₿ Streaming de Bitcoin (QR Capital)](https://github.com/RafaelViniciusBrambillaAlves/qr-capital-data-eng)**  
  Pipeline de dados em tempo real com ingestão, processamento e armazenamento otimizado.  
  Arquitetura baseada em **Kafka + Spark Structured Streaming + Airflow**.  
  Stack: PySpark, Kafka, Airflow, Parquet, Docker
  
- **[ETL Pipeline](https://github.com/RafaelViniciusBrambillaAlves/project)**  
  Pipeline completo de ETL com integração entre múltiplas fontes e sistemas distribuídos.  
  Stack: Python, SQL, PySpark, Airflow, Kafka, Cassandra  
  
- **[Indicium Tech Code Challenge](https://github.com/RafaelViniciusBrambillaAlves/code-challenge/tree/RafaelViniciusBrambillaAlvesv1.0)**: Desafio para vaga de estágio, objetivo é extrair e carregar dados utilizando Python, SQL, Airflow PostgreSQL, Docker e Meltano
